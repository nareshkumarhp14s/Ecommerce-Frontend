# Full Stack E-Commerce Mobile Application (Frontend)

## Overview

A React Native E-Commerce Mobile Application with authentication, product browsing, cart management, wishlist, reviews & ratings, order management, Razorpay payment integration, PDF invoice generation, and admin dashboard support.

## Features

### User Features

* User Registration & Login
* JWT Authentication
* Product Listing
* Product Details
* Search Products
* Wishlist Management
* Shopping Cart
* Address Management
* Reviews & Ratings
* Order Placement
* Razorpay Payment Gateway
* PDF Invoice Download
* Order Tracking

### Admin Features

* Product Management (CRUD)
* User Management
* Order Management
* Analytics Dashboard

## Tech Stack

### Frontend

* React Native CLI
* React Navigation
* Context API
* AsyncStorage
* Axios

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* JWT Authentication
* Razorpay
* Cloudinary

## Installation

```bash
npm install
npx react-native run-android
```
